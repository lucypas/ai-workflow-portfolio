# AI Dataset Readiness Orchestrator

## Purpose

This workflow simulates an AI-assisted dataset onboarding process.

It evaluates whether a dataset is ready for downstream analytics, AI workflows, dashboards, or product use cases.

## Capabilities

- Validates required schema
- Checks null rates for critical fields
- Evaluates dataset freshness
- Applies business rules
- Calculates a readiness score
- Assigns a readiness status
- Generates AI-style recommendations
- Writes an audit log for governance

## Portfolio Positioning

This workflow is useful for explaining how AI can support enterprise data governance, dataset onboarding, and delivery readiness.

It is especially relevant for:
- AI Technical Program Management
- AI Product Management
- Data Platform Programs
- Enterprise Reporting Automation
- RAG / AI Agent Readiness
