# AI Dataset Intake & Validation Gate

## Purpose

This workflow simulates a data-platform intake gate for dataset onboarding.

It validates whether a dataset is ready for downstream analytics, reporting, AI workflows, or product use cases.

## Capabilities

- Checks required columns
- Evaluates null rates for required fields
- Reviews dataset freshness
- Assigns issue severity
- Calculates a readiness score
- Produces a GO / GO_WITH_WARNINGS / NO_GO onboarding decision
- Generates an AI-style review summary

## Portfolio Positioning

This workflow demonstrates how AI-assisted data intake can support enterprise data quality, governance, and delivery readiness.

It is relevant for:
- AI Technical Program Management
- Data Platform Programs
- AI Product Management
- Analytics Enablement
- Enterprise Reporting Automation
